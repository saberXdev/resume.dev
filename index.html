About Me

I strive to build software and UI so remarkable that users don't feel they have one more product, simply one less problem.

I'm a software engineer who's been coding since the Jquery days in 2014, experienced in both front-end, back-end, and Ai web technologies. I build scalable and responsive web applications by leveraging React + Typescript for creating dynamic user interfaces and Next.js, Node.js + Express via RESTful APIs for developing robust server-side components that integrate seamlessly with NoSQL + MySQL databases alongside Langchain + Llama-2 for architecting intelligent and language-aware application layer. Prior to becoming a full-stack developer, I led a vigorous career as a UI / UX designer. Hence Saber X, which signifies "cross"-capabilities in software development and product design for creating killer-app features.

Given the plethora of innovations that we currently have in large language models and the JS ecosystem, you will always find me tinkering with some new framework or LLM from OpenAi, Meta, and HuggingFace as I've grown an addiction to building and living on the cutting-edge of web technologies.

With nearly a decade of code coupled with design, I can single-handedly take projects from ideation to reality.
